## ADT

### Array ADT（陣列）

```c
Create()
Get(i)
Set(i, x)
Insert(i, x)
Delete(i)
Search(x)
Traverse()
```

### Linked List ADT（連結串列）

```c
Create()
Insert(x) / InsertAfter(p, x)
Delete(x)
Search(x)
Traverse()
IsEmpty()
```

### Stack ADT

```c
Create()
Push(x)
Pop()
Top() / Peek()
IsEmpty()
```

### Queue ADT

```c
Create()
Enqueue(x)
Dequeue()
Front()
IsEmpty()
```

### Hash Table ADT

```c
Create()
Hash(key)
Insert(key, value)
Search(key) / Get(key)
Delete(key)
IsEmpty()
```

### Tree ADT

```c
Create()
Insert(node)
Delete(node)
Search(x)
Traverse()
Preorder
Inorder
Postorder
```

### Binary Search Tree (BST) ADT

```c
Search(x)
Insert(x)
Delete(x)
FindMin()
FindMax()
```

### Heap ADT

```c
Insert(x)
DeleteMin() / DeleteMax()
FindMin() / FindMax()
IsEmpty()
```

### Graph ADT

```c
Create()
AddVertex(v)
AddEdge(u, v)
RemoveEdge(u, v)
Adjacent(u, v)
Traverse()
BFS
DFS
```
